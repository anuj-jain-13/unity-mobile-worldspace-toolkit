# World Space Systems

The toolkit provides reusable systems for building scalable world-space gameplay UI in Unity mobile games.

Designed for:
- gameplay HUD elements
- floating rewards
- interactive world-space UI
- scalable gameplay overlays

---

# Included Systems

## WorldObjectPosition
Reusable world-space positioning workflows.

## WorldObjectAutoScaler
Responsive world-space scaling across devices and resolutions.

## StickToPosition
Target-following behavior for gameplay UI elements.

---

# Design Goals

- reusable world-space workflows
- mobile-friendly scaling
- gameplay-oriented UI systems
- maintainable positioning logic
